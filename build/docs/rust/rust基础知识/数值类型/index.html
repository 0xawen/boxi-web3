数值类型

整数​

  • 有符号整数: i8， i16, i32, i64, i128, isize
  • 无符号整数: u8, u16, u32, u64, i128, usize

浮点数​

  • 浮点数: f32, f64

所有权​

所有数值类型都实现了copy 特征。变量赋值的时候， 旧的变量都不会有转移所有权，依然可以使用。
