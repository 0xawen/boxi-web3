# solana workflow

基于 anchor开发的工作流

## 安装anchor

安装相关工具
```bash
cargo install --git https://github.com/project-serum/anchor avm --locked --force

sudo apt-get update && sudo apt-get upgrade && sudo apt-get install -y pkg-config build-essential libudev-dev
avm install latest

avm use latest

anchor --version
```
