# 数组

- 长度固定
- 元素类型相同
- 线性排列



** rust中， 数组是基本类型**



```rust
fn main() {
    // 自动推导 one 类型
    let one = [1,2,3];
    // 显式标注
    let two: [u8; 3] = [1,2,3];
    let blank1 = [0; 3];
    let blank2: [u8; 3] = [0; 3];
    
    let arrays: [[u8, 3]; 4] = [one, two, blank1, blank2];
    
    // 借用 arrays 的元素用作循环
    for a in &arrays {
        for n in a.iter() {
            print!("\t{} + 10 = {}", n, n+10);
        }
    }
}
```
